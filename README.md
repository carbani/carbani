## Install

- netinstall
  - Download packages
    - Common packages
    - Development
    - Fonts
    - Bluetooth
    - Audio
    - Graphic editor
  - Download de / wm
    - Plasma
    - XFCE4
    - Gnome
    - i3wm



#### Pre-installed
- base
- base-devel
- lightdm
- networkmanager
- nvidia
- nvidia-utils
- nvidia-settings

#### Pre-config files
- /usr/lib/NetworkManager/conf.d/20-connectivity.conf
- /etc/systemd/logind.conf
- /etc/xdg/reflector/reflector.conf
